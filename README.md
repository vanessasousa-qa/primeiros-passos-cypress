# My Cypress Automation

This project was developed using Cypress to automate end-to-end tests for key application features, including user login and profile update functionality. Cypress is a JavaScript-based testing framework that allows developers and QA engineers to create reliable automated tests and execute them in a real browser, ensuring a smoother and more efficient validation process.

## Installation
```bash
npm install
```
> **Note:** Node.js must be installed on your machine.

## Running the Tests
### Open Cypress Test Runner (UI)
```bash
npx cypress open
```
### Run tests in headless mode (CI/terminal)
```bash
npx cypress run
```
